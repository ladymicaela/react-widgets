## Background

I wanted to create a collection of simple widgets that I could use as a reference for future projects.

Currently the collection contains:

* A Clock
* Local Weather
* Tabs
* Autocomplete
* Drag and Drog
* Calculator
* Magic 8 Ball
* Dice Roller
* Notes
* Countdown
